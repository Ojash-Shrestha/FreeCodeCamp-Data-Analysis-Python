# FreeCodeCamp-Data-Analysis-Python
## About this repository
5 data analysis projects completed as part of the freeCodeCamp
Scientific Computing with Python certification. Each project
demonstrates practical data manipulation, statistical analysis,
and visualization using Python's core data science stack.

## Projects

| # | Project | Key skills | Dataset |
|---|---------|-----------|---------|
| 1 | Mean-Variance-Std Calculator | NumPy, statistics | Generated |
| 2 | Demographic Data Analyzer | pandas, groupby, filtering | UCI Adult |
| 3 | Medical Data Visualizer | seaborn, matplotlib, heatmaps | Medical exams |
| 4 | Page View Time Series | time series, resampling, box plots | FCC Forum |
| 5 | Sea Level Predictor | linear regression, scipy, prediction | EPA |

## Tech stack
Python 3.11 · pandas · NumPy · matplotlib · seaborn · scipy
